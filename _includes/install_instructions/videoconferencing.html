Configure seu espaço de trabalho

Esse minicurso segue o modelo de live-coding do The Carpentries, ou seja, programação em tempo real acompanhando os instrutores. Para isso, você precisará ter na tela tanto a janela do programa que vamos usar (RStudio, Excel etc.) como a janela do Zoom. Para ver as duas ao mesmo tempo, recomendamos uma das seguintes opções:

  • Dois monitores: Se você tem dois monitores disponíveis, planeje para manter o software de interesse em um dos monitores, e a janela do Zoom no outro.
  • Dois aparelhos: Se um segundo monitor não está disponível, você tem algum outro aparelho (tablet, celular) com uma tela de tamanho razoável? Caso sim, tente usar a tela menor para o vídeo e a tela do computador para acompanhar o programa sobre o qual vamos aprender.
  • Divida sua tela: Se o computador com uma tela é a única opção disponível, pratique ter as duas janelas abertas ao mesmo tempo (a janela do Zoom e a do RStudio). Qual a melhor disposição das duas janelas na sua tela? Seria melhor alternar entre as duas janelas com um atalho do teclado? Teste com antencedência para encontrar o melhor ajuste para você, e para que sua experiência durante o minicurso seja a melhor possível.
